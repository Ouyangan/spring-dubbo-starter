# spring-dubbo-starter
spring+dubbo脚手架

# 启动
- 将服务提供方和消费方的注册中心地址修改为你自己的地址 <dubbo:registry address="zookeeper://10.120.44.79:2181"/>
- 运行provider
- 运行consumer
